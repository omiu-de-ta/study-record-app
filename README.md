## 概要
学習内容をシンプルに記録・管理できるWebアプリです。  
「記録が面倒で続かない」という課題を解決するため、最小限の操作で使える設計にしました。
クリックで編集できるなど、直感的に操作できるUIを意識しています。

## 作成した背景
日々の学習の中で、記録を残すことで復習しやすくなり、自分の成長を客観的に振り返ることができると感じました。  
しかし、記録する手間が負担となり、継続できないという課題がありました。  
そこで、最小限の操作で簡単に記録できるシンプルなアプリを作成しました。

## 機能
- 学習内容の追加
- 学習内容の一覧表示
- 学習内容の削除
- 学習内容の編集（クリックで編集モードへ）
- 学習日付の自動記録
- localStorageによるデータ永続化（リロード後も保持）

## 使い方
1. 入力欄に学習内容を入力
2.「追加」ボタンをクリック
3. 一覧に記録が表示される
4. 内容をクリックすると編集可能
5. 「削除」ボタンで削除
    
## 使用技術
- HTML
- CSS
- JavaScript（Vanilla JS）
- localStorage

## 工夫した点
- クリックで編集できる直感的なUIを実装
- 学習日付を自動記録し、振り返りしやすい設計にした
- 入力項目を最小限にし、継続しやすいシンプルな操作性を実現
- 空入力を防ぐバリデーションを追加し、入力ミスを防止

## 設計・実装のポイント
- render関数でUI更新を一元化し、状態と表示の不整合を防止
- editingIdで編集状態を管理し、追加処理と編集処理の責務を分離
- map / filterを使用し、副作用の少ないイミュータブルなデータ更新を実装
- Date.now()で一意なIDを生成し、削除・編集の正確性を担保
- localStorageからの読み込み・保存処理を実装し、データの永続化を実現
- CSSを分離し、見た目とロジックの分離による保守性を向上
  
## 苦労した点
削除機能の実装時に、意図しないデータが削除される問題がありました。  
IDで管理する設計に変更することで解決しました。

## 学んだこと
- JavaScriptでCRUD処理（追加・削除・編集）を実装できるようになった
- map / filterを使ったデータ操作に慣れてきた
- localStorageを使ったデータ保存の基本を理解した
- まだ理解が浅い部分もあるが、DOM操作と再描画の流れを実装できた
